# Use Case: Manage Tags

**Actor**: Admin
**Module**: `application.manage_tags`

## Goal

Administrator manages the catalog of competition tags that organizers can attach to competitions.

## Interactors

- [CreateCompetitionTag](./create_tag.md) — create a new tag
- [ReadCompetitionTag](./read_tag.md) — read a tag by ID
- [ListCompetitionTags](./list_tags.md) — search and paginate tags
- [DeleteCompetitionTag](./delete_tag.md) — delete a tag
