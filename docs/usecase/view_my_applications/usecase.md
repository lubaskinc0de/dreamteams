# Use Case: View My Applications

**Actor**: Participant
**Module**: `application.view_my_applications`

## Goal

A participant reads a single submitted application and browses their application history.

## Interactors

- [ReadMyApplication](./read_my_application.md) — read a single application submitted by the current participant
- [ListMyApplications](./list_my_applications.md) — list all applications submitted by the current participant
