# Use Case: Manage Profile

**Actor**: User
**Module**: `application.manage_profile`

## Goal

User manages their profile: view information, update assigned role data, and delete their own profile.

## Interactors

- [ReadProfile](./read_profile.md) — read current user profile
- [UpdateParticipant](./update_participant.md) — update participant profile
- [UpdateOrganizer](./update_organizer.md) — update organizer profile
- [DeleteProfile](./delete_profile.md) — delete current user profile
