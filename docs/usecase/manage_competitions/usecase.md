# Use Case: Manage Competitions

**Actor**: Organizer
**Module**: `application.manage_competitions`

## Goal

Organizer manages their existing competitions: view details, update information, and delete competitions.

## Interactors

- [ListCompetitions](./list_competitions.md) — list all organizer's competitions
- [ReadCompetition](./read_competition.md) — read competition by ID
- [UpdateCompetition](./update_competition.md) — update competition data
- [DeleteCompetition](./delete_competition.md) — delete competition
