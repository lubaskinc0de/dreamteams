# Domain

## Purpose

Represents IT domain area for competitions and participant skills.

## Type

Value object (Enum)

## Values

| Value | Description |
|-------|-------------|
| `FRONTEND` | Frontend development (HTML, CSS, JavaScript, React, Vue, etc.) |
| `MOBILE` | Mobile development (iOS, Android, React Native, Flutter) |
| `BACKEND` | Backend development (APIs, databases, server-side logic) |
| `AI` | Artificial Intelligence and Machine Learning |
| `DEVOPS` | DevOps, infrastructure, CI/CD, cloud platforms |

## Business Rules

1. Competition must have at least one domain
2. Participant skills are tagged with domains
3. Used for matching participants with relevant competitions
