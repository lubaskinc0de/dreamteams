# Entities

Core domain models of the DreamTeams platform. Entities contain identity, state, and domain business rules.

| Entity | Description |
|--------|-------------|
| [User](user.md) | Base account — container for organizer/participant roles |
| [Organizer](organizer.md) | Organization profile linked to a User |
| [Participant](participant.md) | Individual profile linked to a User |
| [Competition](competition.md) | Hackathon or olympiad event |
| [CompetitionTag](competition-tag.md) | Searchable tag catalog entry used by competitions |
| [Application](application.md) | Participant's application to a competition |
| [ApplicationForm](application-form.md) | Optional extra fields attached to a competition |
| [OrganizerInvite](organizer-invite.md) | Invite code for organizer registration |
